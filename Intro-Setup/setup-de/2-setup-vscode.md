# Installation von Visual Studio Code (VS Code)

Dieser Leitfaden hilft dir, Visual Studio Code auf Windows, macOS und Linux zu installieren.

---

## 1. VS Code herunterladen

1. Gehe zur offiziellen VS Code Website: [https://code.visualstudio.com](https://code.visualstudio.com)
2. Klicke auf den Download-Button für dein Betriebssystem:

   * **Windows** → User Installer (empfohlen) oder System Installer
   * **macOS** → `.dmg`-Datei
   * **Linux** → `.deb` (Debian/Ubuntu) oder `.rpm` (Fedora/Red Hat)

---

## 2. VS Code installieren

### Windows

1. Öffne die heruntergeladene `.exe`-Datei.
2. Folge dem Installationsassistenten:

   * Akzeptiere die Lizenzvereinbarung.
   * Wähle den Installationsort.
   * Wähle optionale Aufgaben (z. B. „Öffnen mit Code“ Kontextmenü hinzufügen).
3. Klicke auf **Installieren** und warte, bis die Installation abgeschlossen ist.
4. Klicke auf **Fertigstellen**, um VS Code zu starten.

### macOS

1. Öffne die heruntergeladene `.dmg`-Datei.
2. Ziehe das **Visual Studio Code**-Symbol in den **Programme**-Ordner.
3. Starte VS Code aus dem Programme-Ordner oder über die Spotlight-Suche.

### Linux (Debian/Ubuntu)

```bash
sudo apt update
sudo apt install ./<file>.deb
```
