# Installing Visual Studio Code (VS Code)

This guide will help you install Visual Studio Code on Windows, macOS, and Linux.

---

## 1. Download VS Code

1. Go to the official VS Code website: [https://code.visualstudio.com](https://code.visualstudio.com)
2. Click on the download button for your operating system:
   - **Windows** → User Installer (recommended) or System Installer
   - **macOS** → `.dmg` file
   - **Linux** → `.deb` (Debian/Ubuntu) or `.rpm` (Fedora/Red Hat)

---

## 2. Install VS Code

### Windows

1. Open the downloaded `.exe` file.
2. Follow the installation wizard:
   - Accept the license agreement.
   - Choose an installation location.
   - Select optional tasks (e.g., add “Open with Code” context menu).
3. Click **Install** and wait for the installation to complete.
4. Click **Finish** to launch VS Code.

### macOS

1. Open the downloaded `.dmg` file.
2. Drag the **Visual Studio Code** icon to the **Applications** folder.
3. Launch VS Code from the Applications folder or Spotlight Search.
